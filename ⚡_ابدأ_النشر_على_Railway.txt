الخطوة 1️⃣: إنشاء حساب على Railway
───────────────────────────

1. افتح: https://railway.app

2. اضغط "Login"

3. سجل الدخول بـ GitHub (موصى به!)

4. ✅ تم!


الخطوة 2️⃣: رفع المشروع على GitHub
───────────────────────────

إذا كان المشروع موجود على GitHub:
   ✅ انتقل للخطوة 3

إذا لم يكن موجود:

1. افتح PowerShell:
   cd C:\Users\DELL\DED

2. شغل الأوامر:
   git init
   git add .
   git commit -m "Deploy to Railway"

3. أنشئ Repository على GitHub:
   https://github.com/new

4. اربط المشروع:
   git remote add origin https://github.com/YOUR_USERNAME/ded-erp.git
   git push -u origin main


الخطوة 3️⃣: إنشاء مشروع على Railway
───────────────────────────

1. اذهب إلى: https://railway.app/new

2. اضغط "Deploy from GitHub repo"

3. اختر Repository الخاص بك

4. ✅ Railway سيبدأ البناء تلقائياً!


الخطوة 4️⃣: إضافة قاعدة بيانات
───────────────────────────

1. في صفحة المشروع، اضغط "+ New"

2. اختر "Database" → "Add PostgreSQL"

3. ✅ تم! Railway سيربطها تلقائياً


الخطوة 5️⃣: إعداد المتغيرات
───────────────────────────

1. في صفحة المشروع → "Variables"

2. أضف:
   FLASK_ENV=production
   SECRET_KEY=your-super-secret-key-123456

3. ⚠️ غير SECRET_KEY إلى قيمة عشوائية!


الخطوة 6️⃣: تهيئة قاعدة البيانات
───────────────────────────

1. في Railway → "Console" أو "Terminal"

2. شغل:
   python init_production_db.py

3. ✅ سيتم إنشاء الجداول!


الخطوة 7️⃣: الحصول على الرابط
───────────────────────────

1. في صفحة المشروع → "Settings"

2. ابحث عن "Domains"

3. اضغط "Generate Domain"

4. ✅ ستحصل على رابط مثل:
   https://ded-erp-production.up.railway.app

المشكلة: Build Failed
───────────────────────────

الحل:
1. تأكد من وجود requirements.txt
2. تأكد من وجود railway.json
3. تحقق من Logs في Railway


المشكلة: Database Error
───────────────────────────

الحل:
1. تأكد من إضافة PostgreSQL
2. شغل: python init_production_db.py


المشكلة: 502 Bad Gateway
───────────────────────────

الحل:
1. تحقق من Logs في Railway
2. تأكد من أن Port صحيح
