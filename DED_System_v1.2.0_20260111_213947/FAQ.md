# الأسئلة الشائعة (FAQ)

## عام

### ما هو نظام إدارة المخزون المتكامل؟
نظام ERP شامل لإدارة المخزون والمبيعات والمشتريات والمحاسبة والموارد البشرية، مبني بلغة Python باستخدام Flask.

### هل النظام مجاني؟
نعم، النظام مفتوح المصدر ومرخص تحت MIT License، مما يعني أنه مجاني للاستخدام الشخصي والتجاري.

### هل يدعم النظام اللغة العربية؟
نعم، النظام مصمم بالكامل للغة العربية مع دعم RTL (من اليمين لليسار).

### هل يمكن استخدامه في الإنتاج؟
نعم، لكن تأكد من:
- تغيير كلمة المرور الافتراضية
- استخدام قاعدة بيانات قوية (PostgreSQL/MySQL)
- تفعيل HTTPS
- أخذ نسخ احتياطية منتظمة

## التثبيت والتشغيل

### ما هي متطلبات التشغيل؟
- Python 3.8 أو أحدث
- 2GB RAM (الحد الأدنى)
- 500MB مساحة تخزين

### كيف أقوم بتثبيت النظام؟
راجع [INSTALLATION.md](INSTALLATION.md) للتعليمات التفصيلية.

### لا يعمل التطبيق، ماذا أفعل؟
1. تأكد من تثبيت Python بشكل صحيح
2. تأكد من تفعيل البيئة الافتراضية
3. تأكد من تثبيت جميع المتطلبات: `pip install -r requirements.txt`
4. تحقق من السجلات (logs) للأخطاء

### كيف أغير المنفذ (Port)؟
في ملف `run.py`، غيّر:
```python
app.run(host='0.0.0.0', port=5001, debug=True)
```

## قاعدة البيانات

### ما نوع قاعدة البيانات المستخدمة؟
افتراضياً SQLite، لكن يمكن استخدام PostgreSQL أو MySQL.

### كيف أغير قاعدة البيانات إلى PostgreSQL؟
1. ثبّت PostgreSQL
2. في ملف `.env`، غيّر:
   ```
   DATABASE_URL=postgresql://username:password@localhost/dbname
   ```
3. ثبّت المتطلبات: `pip install psycopg2-binary`
4. أعد تهيئة قاعدة البيانات: `flask init-db`

### كيف أخذ نسخة احتياطية؟
```bash
# SQLite
cp erp_system.db backups/backup_$(date +%Y%m%d).db

# PostgreSQL
pg_dump dbname > backup.sql
```

### كيف أستعيد نسخة احتياطية؟
```bash
# SQLite
cp backups/backup_20260110.db erp_system.db

# PostgreSQL
psql dbname < backup.sql
```

## الاستخدام

### كيف أضيف مستخدم جديد؟
1. سجّل دخول كمدير
2. اذهب إلى: الإعدادات > المستخدمين
3. اضغط "إضافة مستخدم جديد"
4. أدخل البيانات واختر الدور

### كيف أضيف منتج جديد؟
1. المخزون > المنتجات
2. اضغط "إضافة منتج جديد"
3. أدخل البيانات
4. احفظ

### كيف أنشئ فاتورة بيع؟
1. المبيعات > الفواتير
2. اضغط "إضافة فاتورة جديدة"
3. اختر العميل
4. أضف المنتجات
5. احفظ

### كيف أطبع فاتورة؟
اضغط على زر "طباعة" في صفحة الفاتورة، أو استخدم Ctrl+P.

## الصلاحيات

### ما هي الأدوار المتاحة؟
- **مدير النظام (Admin):** صلاحيات كاملة
- **مدير (Manager):** معظم الصلاحيات
- **مستخدم (User):** صلاحيات محدودة

### كيف أغير صلاحيات مستخدم؟
1. الإعدادات > المستخدمين
2. اضغط "تعديل" على المستخدم
3. غيّر الدور
4. احفظ

### نسيت كلمة المرور، ماذا أفعل؟
إذا كنت المدير الوحيد:
1. افتح قاعدة البيانات
2. احذف المستخدم القديم
3. أعد تشغيل `flask init-db` لإنشاء مدير جديد

## التقارير

### كيف أطبع تقرير؟
1. اذهب إلى التقارير
2. اختر التقرير المطلوب
3. حدد الفترة الزمنية
4. اضغط "عرض"
5. اضغط "طباعة" أو Ctrl+P

### كيف أصدّر تقرير إلى Excel؟
هذه الميزة قيد التطوير وستتوفر في الإصدار 1.1.

### التقارير فارغة، لماذا؟
تأكد من:
- وجود بيانات في الفترة المحددة
- اختيار الفترة الزمنية الصحيحة
- وجود صلاحيات لعرض التقرير

## الأداء

### النظام بطيء، كيف أحسّنه؟
1. استخدم PostgreSQL بدلاً من SQLite
2. فعّل التخزين المؤقت (Caching)
3. استخدم خادم قوي

### كم عدد المستخدمين المتزامنين المدعومين؟
- SQLite: 10-20 مستخدم
- PostgreSQL: 100+ مستخدم
