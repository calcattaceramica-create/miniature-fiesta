                      🔄 التحديث التلقائي - Auto-Update
═══════════════════════════════════════════════════════════════════════════════

الإحصائيات تتحدث تلقائياً عند:
Statistics auto-update when:

  ✓ إضافة ترخيص جديد        → Adding a new license
  ✓ حذف ترخيص               → Deleting a license
  ✓ تفعيل ترخيص             → Activating a license
  ✓ إيقاف ترخيص مؤقتاً       → Suspending a license
  ✓ الضغط على زر التحديث     → Clicking refresh button
