الخطوة 1️⃣: إنشاء حساب
───────────────────────────

1. افتح: https://render.com

2. اضغط "Get Started for Free"

3. سجل الدخول بـ GitHub (موصى به!)

4. ✅ تم!


الخطوة 2️⃣: ربط GitHub
───────────────────────────

1. في Render Dashboard → "New +"

2. اختر "Web Service"

3. اضغط "Connect GitHub"

4. اختر Repository:
   calcattaceramica-create/miniature-fiesta

5. ✅ تم الربط!


الخطوة 3️⃣: إعداد المشروع
───────────────────────────

Render سيكتشف render.yaml تلقائياً! ✅

فقط تأكد من:
   - Name: ded-erp-system
   - Region: Frankfurt
   - Branch: main

اضغط "Create Web Service"


الخطوة 4️⃣: إضافة قاعدة البيانات
───────────────────────────

1. في Dashboard → "New +"

2. اختر "PostgreSQL"

3. Name: ded-database
   Region: Frankfurt
   Plan: Free ✅

4. اضغط "Create Database"


الخطوة 5️⃣: ربط قاعدة البيانات
───────────────────────────

1. ارجع إلى Web Service

2. اذهب إلى "Environment"

3. أضف:
   Key: DATABASE_URL
   Value: (انسخه من صفحة Database)

4. اضغط "Save Changes"


الخطوة 6️⃣: تهيئة قاعدة البيانات
───────────────────────────

1. في Web Service → "Shell"

2. شغل:
   python init_production_db.py

3. ✅ تم!
