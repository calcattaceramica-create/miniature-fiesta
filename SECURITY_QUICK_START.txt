                        ⚡  البدء السريع (5 دقائق)

═══════════════════════════════════════════════════════════════════════════════════


┌───────────────────────────────────────────────────────────────────────────────┐
│  1️⃣  تغيير كلمة المرور الافتراضية (إلزامي!)                                 │
└───────────────────────────────────────────────────────────────────────────────┘

  ⚠️  كلمة المرور الافتراضية:
      Username: admin
      Password: admin123

  ✅  خطوات التغيير:
      1. سجل دخول بالحساب الافتراضي
      2. اذهب إلى: القائمة > تغيير كلمة المرور
      3. أدخل كلمة مرور قوية (8+ أحرف، حرف كبير، رقم)
      4. احفظ


┌───────────────────────────────────────────────────────────────────────────────┐
│  2️⃣  إنشاء حسابات للمستخدمين                                                │
└───────────────────────────────────────────────────────────────────────────────┘

  ✅  خطوات الإنشاء:
      1. اذهب إلى: القائمة > الإعدادات > المستخدمون
      2. اضغط "إضافة مستخدم جديد"
      3. املأ البيانات
      4. اختر الصلاحيات المناسبة
      5. فعّل "يجب تغيير كلمة المرور عند أول دخول"
      6. احفظ


┌───────────────────────────────────────────────────────────────────────────────┐
│  3️⃣  مراجعة لوحة التحكم الأمنية                                             │
└───────────────────────────────────────────────────────────────────────────────┘

  ✅  الوصول:
      القائمة > الأمان > لوحة التحكم

  📊  ما ستراه:
      - إجمالي السجلات الأمنية
      - محاولات الدخول الفاشلة اليوم
      - الجلسات النشطة
      - الحسابات المقفلة
      - الأحداث الأخيرة


═══════════════════════════════════════════════════════════════════════════════════

                        🛡️  الميزات الأمنية المفعّلة

═══════════════════════════════════════════════════════════════════════════════════


  ✅  حماية ضد Brute Force:
      ═══════════════════════════
      - قفل الحساب بعد 5 محاولات فاشلة
      - مدة القفل: 30 دقيقة
      - تسجيل جميع المحاولات


  ✅  أمان كلمات المرور:
      ═══════════════════════
      - الحد الأدنى: 8 أحرف
      - يجب حرف كبير واحد
      - يجب رقم واحد
      - تشفير قوي (PBKDF2-SHA256)


  ✅  أمان الجلسات:
      ═══════════════
      - انتهاء الجلسة بعد ساعتين
      - حماية CSRF
      - HttpOnly Cookies
      - تسجيل جميع الجلسات


  ✅  تسجيل الأنشطة:
      ═══════════════════
      - تسجيل محاولات الدخول
      - تسجيل تغييرات كلمة المرور
      - تسجيل عناوين IP
      - مستويات خطورة مختلفة


  ✅  حماية الشبكة:
      ═══════════════
      - Rate Limiting (تحديد معدل الطلبات)
      - قائمة بيضاء للـ IP
      - حماية من DDOS


═══════════════════════════════════════════════════════════════════════════════════

                        📋  المهام اليومية للمسؤول

═══════════════════════════════════════════════════════════════════════════════════


  🔍  كل صباح (5 دقائق):
      ═══════════════════════
      1. افتح لوحة التحكم الأمنية
      2. راجع محاولات الدخول الفاشلة
      3. تحقق من الجلسات النشطة
      4. راجع الأحداث المشبوهة


  📊  كل أسبوع (15 دقيقة):
      ═══════════════════════
      1. راجع سجلات الأمان الكاملة
      2. راجع قائمة IP البيضاء
      3. راجع الحسابات النشطة
      4. راجع الصلاحيات


  💾  كل شهر (30 دقيقة):
      ═══════════════════════
      1. نسخ احتياطي لقاعدة البيانات
      2. اختبار استعادة النسخة الاحتياطية
      3. مراجعة شاملة للأمان
      4. تحديث كلمات المرور


═══════════════════════════════════════════════════════════════════════════════════

                        ⚠️  تنبيهات أمنية مهمة

═══════════════════════════════════════════════════════════════════════════════════


  🔴  لا تفعل أبداً:
      ═══════════════
      ❌ مشاركة كلمة المرور
      ❌ استخدام كلمات مرور ضعيفة
      ❌ ترك الجهاز مفتوحاً
      ❌ تجاهل التنبيهات الأمنية
      ❌ منح صلاحيات غير ضرورية


  🟢  افعل دائماً:
      ═══════════════
      ✅ استخدم كلمات مرور قوية
      ✅ سجّل خروج عند الانتهاء
      ✅ راجع السجلات بانتظام
      ✅ حدّث النظام بانتظام
      ✅ احتفظ بنسخ احتياطية

                        🔧  حل المشاكل الشائعة

═══════════════════════════════════════════════════════════════════════════════════


  ❓  حساب مقفل؟
      ═══════════
      الحل 1: انتظر 30 دقيقة
      الحل 2: اتصل بالمسؤول لفتح الحساب من:
              القائمة > الأمان > لوحة التحكم
