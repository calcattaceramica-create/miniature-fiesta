# نظام إدارة الموارد البشرية (HR System)

## نظرة عامة
نظام شامل لإدارة الموارد البشرية يتضمن إدارة الموظفين، الحضور والانصراف، الإجازات، والرواتب.

## المميزات الرئيسية

### 1. إدارة الموظفين
- إضافة وتعديل بيانات الموظفين
- تتبع معلومات الموظفين الشخصية والوظيفية
- إدارة الأقسام والمناصب
- عرض تفاصيل كل موظف

### 2. إدارة الحضور والانصراف
- تسجيل حضور وانصراف الموظفين
- حساب ساعات العمل تلقائياً
- تتبع التأخير والغياب
- حساب الساعات الإضافية

### 3. إدارة الإجازات
- طلب إجازات جديدة
- الموافقة أو رفض طلبات الإجازات
- أنواع إجازات متعددة (سنوية، مرضية، طارئة، إلخ)
- تتبع رصيد الإجازات لكل موظف

### 4. إدارة الرواتب
- إنشاء كشوف رواتب شهرية
- حساب الراتب الأساسي والبدلات
- حساب الخصومات والساعات الإضافية
- اعتماد ودفع الرواتب

### 5. التقارير
- تقرير الحضور والانصراف
- تقرير الموظفين
- تقرير الإجازات
- تقرير الرواتب
- تقارير الأقسام والمناصب

## هيكل قاعدة البيانات

### الجداول الرئيسية:
1. **employees** - بيانات الموظفين
2. **departments** - الأقسام
3. **positions** - المناصب الوظيفية
4. **attendance** - سجلات الحضور والانصراف
5. **leaves** - طلبات الإجازات
6. **leave_types** - أنواع الإجازات
7. **payroll** - كشوف الرواتب

## الصفحات والواجهات

### صفحات الموظفين:
- `/hr/employees` - قائمة الموظفين
- `/hr/add_employee` - إضافة موظف جديد
- `/hr/edit_employee/<id>` - تعديل بيانات موظف
- `/hr/employee_details/<id>` - تفاصيل الموظف

### صفحات الحضور:
- `/hr/attendance` - سجلات الحضور
- `/hr/add_attendance` - تسجيل حضور

### صفحات الإجازات:
- `/hr/leaves` - طلبات الإجازات
- `/hr/add_leave` - طلب إجازة جديدة
- `/hr/leave_types` - أنواع الإجازات

### صفحات الرواتب:
- `/hr/payroll` - كشوف الرواتب
- `/hr/generate_payroll` - إنشاء كشف رواتب
- `/hr/payroll_details/<id>` - تفاصيل الراتب

### صفحات أخرى:
- `/hr/dashboard` - لوحة التحكم
- `/hr/departments` - الأقسام
- `/hr/positions` - المناصب
- `/hr/reports` - التقارير

## التثبيت والتشغيل

### 1. تثبيت المتطلبات:
```bash
pip install -r requirements.txt
```

### 2. إنشاء قاعدة البيانات:
```bash
python init_db.py
```

### 3. تشغيل التطبيق:
```bash
python run.py
```

### 4. الوصول إلى النظام:
افتح المتصفح على: `http://localhost:5000`

## الاستخدام

### إضافة موظف جديد:
1. اذهب إلى "الموظفين" > "إضافة موظف"
2. املأ البيانات المطلوبة
3. اضغط "حفظ"

### تسجيل حضور:
1. اذهب إلى "الحضور والانصراف" > "تسجيل حضور"
2. اختر الموظف
3. حدد نوع السجل (حضور/انصراف)
4. اضغط "حفظ"

### طلب إجازة:
1. اذهب إلى "الإجازات" > "طلب إجازة"
2. اختر الموظف ونوع الإجازة
3. حدد التواريخ والسبب
4. اضغط "إرسال الطلب"

### إنشاء كشف رواتب:
1. اذهب إلى "الرواتب" > "إنشاء كشف رواتب"
2. اختر الشهر والسنة
3. اضغط "إنشاء كشف الرواتب"
4. راجع الكشف واعتمده
5. سجل الدفع

## الملاحظات المهمة

- يتم حساب الساعات الإضافية تلقائياً من سجلات الحضور
- يجب اعتماد كشف الرواتب قبل تسجيل الدفع
- يمكن الموافقة أو رفض طلبات الإجازات من صفحة الإجازات
- جميع التقارير قابلة للطباعة
