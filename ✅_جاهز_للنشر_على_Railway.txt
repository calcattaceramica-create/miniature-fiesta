الخطوة 1️⃣: افتح الدليل التفاعلي
───────────────────────────

افتح الملف التالي في المتصفح:
   📂 🎯_Railway_دليل_تفاعلي.html

أو اقرأ الدليل النصي:
   📂 ⚡_ابدأ_النشر_على_Railway.txt


الخطوة 2️⃣: إنشاء حساب على Railway
───────────────────────────

1. افتح: https://railway.app

2. اضغط "Login"

3. سجل الدخول بـ GitHub

4. ✅ تم!


الخطوة 3️⃣: إنشاء مشروع جديد
───────────────────────────

1. اذهب إلى: https://railway.app/new

2. اضغط "Deploy from GitHub repo"

3. اختر Repository:
   calcattaceramica-create/miniature-fiesta

4. ✅ Railway سيبدأ البناء تلقائياً!


الخطوة 4️⃣: إضافة قاعدة بيانات
───────────────────────────

1. في صفحة المشروع، اضغط "+ New"

2. اختر "Database" → "Add PostgreSQL"

3. ✅ تم!


الخطوة 5️⃣: إعداد المتغيرات
───────────────────────────

1. في صفحة المشروع → "Variables"

2. أضف:
   FLASK_ENV=production
   SECRET_KEY=change-this-to-random-string-123456789


الخطوة 6️⃣: تهيئة قاعدة البيانات
───────────────────────────

1. في Railway → "Console"

2. شغل:
   python init_production_db.py


الخطوة 7️⃣: الحصول على الرابط
───────────────────────────

1. في صفحة المشروع → "Settings"

2. ابحث عن "Domains"

3. اضغط "Generate Domain"

4. ✅ ستحصل على رابط!
