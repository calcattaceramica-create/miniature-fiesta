🌟 الطريقة 2: PythonAnywhere (يعمل 24/7)
═══════════════════════════════════════════════════════════════════════════════

إذا كنت تريد رابط يعمل حتى بعد إغلاق الكمبيوتر:


الخطوة 1️⃣: إنشاء حساب
───────────────────────────

1. افتح: https://www.pythonanywhere.com/registration/register/beginner/

2. املأ البيانات:
   - Username: اختر اسم (مثل: dedapp)
   - Email: بريدك الإلكتروني
   - Password: كلمة مرور قوية

3. اضغط "Register"

4. افتح بريدك الإلكتروني وفعّل الحساب


الخطوة 2️⃣: رفع المشروع
───────────────────────────

1. بعد تسجيل الدخول، اضغط "Files"

2. اضغط "Upload a file"

3. ارفع هذه الملفات:
   - run.py
   - requirements.txt
   - app/ (المجلد كامل)

4. أو استخدم Git:
   - اضغط "Consoles" → "Bash"
   - اكتب:
     git clone https://github.com/calcattaceramica-create/miniature-fiesta.git


الخطوة 3️⃣: تثبيت المكتبات
───────────────────────────

1. في Bash Console، اكتب:
   cd miniature-fiesta
   pip install -r requirements.txt


الخطوة 4️⃣: إنشاء Web App
───────────────────────────

1. اضغط "Web" من القائمة العلوية

2. اضغط "Add a new web app"

3. اختر "Flask"

4. اختر Python 3.10

5. في "Source code"، اكتب:
   /home/dedapp/miniature-fiesta

6. في "WSGI configuration file"، افتحه وعدّل:
   from run import app as application


الخطوة 5️⃣: احصل على الرابط!

🚀 الطريقة 3: Render.com (الأفضل!)
═══════════════════════════════════════════════════════════════════════════════

الخطوة 1️⃣: إنشاء حساب
───────────────────────────

1. افتح: https://render.com

2. اضغط "Get Started for Free"

3. اضغط "Sign in with GitHub"

4. أدخل بيانات GitHub

5. اضغط "Authorize Render"


الخطوة 2️⃣: إنشاء Web Service
───────────────────────────

1. اضغط "New +" في الأعلى

2. اختر "Web Service"

3. اضغط "Connect GitHub"

4. ابحث عن: calcattaceramica-create/miniature-fiesta

5. اضغط "Connect"


الخطوة 3️⃣: إعداد المشروع
───────────────────────────

✅ Render سيكتشف ملف render.yaml تلقائياً!

فقط تأكد من:
- Name: ded-erp-system
- Region: Frankfurt
- Branch: main

ثم اضغط "Create Web Service"


الخطوة 4️⃣: انتظر البناء
───────────────────────────

1. ستظهر "Deploying..."
2. انتظر 3-5 دقائق
3. ستظهر "Live" ✅
