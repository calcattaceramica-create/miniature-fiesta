                    🔑 خطوات الحصول على API Token
                    (دقيقة واحدة فقط!)
═══════════════════════════════════════════════════════════════════════════════


الخطوة 1️⃣: سجل الدخول إلى Render.com
───────────────────────────────────────────

1. افتح: https://dashboard.render.com/register
2. اضغط "Sign in with GitHub"
3. أدخل بيانات GitHub
4. اضغط "Authorize Render"


الخطوة 2️⃣: افتح صفحة API Keys
───────────────────────────────────────────

1. بعد تسجيل الدخول، افتح:
   👉 https://dashboard.render.com/u/settings/api-keys

2. أو:
   - اضغط على صورة ملفك الشخصي (في الأعلى)
   - اختر "Account Settings"
   - اختر "API Keys" من القائمة الجانبية


الخطوة 3️⃣: إنشاء API Key جديد
───────────────────────────────────────────

1. اضغط "Create API Key"

2. أدخل اسم للـ Key:
   Name: DED-ERP-Deploy

3. اضغط "Create API Key"


الخطوة 4️⃣: انسخ الـ Token
───────────────────────────────────────────

⚠️ مهم جداً:
   - سيظهر لك الـ Token مرة واحدة فقط!
   - انسخه فوراً!
   - سيبدأ بـ: rnd_...

مثال:
   rnd_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx


الخطوة 5️⃣: أعطني الـ Token
───────────────────────────────────────────

الصق الـ Token هنا في المحادثة:

rnd_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx


═══════════════════════════════════════════════════════════════════════════════
                    ⚠️ ملاحظات مهمة
═══════════════════════════════════════════════════════════════════════════════

✅ الـ Token آمن معي - لن أحفظه أو أشاركه
✅ يمكنك حذف الـ Token بعد التثبيت
✅ الـ Token يعطيني صلاحية إنشاء Web Service فقط


═══════════════════════════════════════════════════════════════════════════════
                    🚀 بعد إعطائي الـ Token

سأقوم بـ:
1. تثبيت Render CLI
2. تسجيل الدخول باستخدام الـ Token
3. إنشاء Web Service تلقائياً
4. إنشاء قاعدة البيانات تلقائياً
5. ربط كل شيء تلقائياً
6. إعطائك الرابط النهائي!
