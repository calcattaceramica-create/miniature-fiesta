✅ واجهة رسومية احترافية

   1️⃣ إنشاء ترخيص جديد

      - اختيار نوع الترخيص

      - الحصول على مفتاح الترخيص فوراً
   
   2️⃣ عرض جميع التراخيص
      - جدول شامل بجميع التراخيص
      - عرض الحالة والأيام المتبقية
      - تحديث فوري
   
   3️⃣ إدارة الترخيص
      - عرض تفاصيل كاملة
      - تعليق/إلغاء تعليق
      - تمديد الترخيص
      - إلغاء نهائي

خطوة 1: افتح التطبيق
────────────────────

    انقر نقراً مزدوجاً على: License_Manager_GUI.pyw


خطوة 2: أنشئ ترخيص
────────────────────

    1. اذهب لتبويب "إنشاء ترخيص جديد"
    2. املأ البيانات:
       - اسم العميل ✅
       - اسم المستخدم ✅
       - كلمة المرور ✅
    3. اضغط "إنشاء الترخيص"
    4. احفظ المعلومات! 📝


خطوة 3: شاهد التراخيص
────────────────────────

    1. اذهب لتبويب "عرض التراخيص"
    2. شاهد جميع التراخيص في جدول
    3. اضغط "تحديث" للتحديث


خطوة 4: أدر الترخيص
────────────────────

    1. اذهب لتبويب "إدارة الترخيص"
    2. أدخل رقم الترخيص
    3. اضغط "تحميل"
    4. استخدم الأزرار:
       - ⏸️ تعليق
       - ▶️ إلغاء التعليق
       - 📅 تمديد
       - ❌ إلغاء نهائي
