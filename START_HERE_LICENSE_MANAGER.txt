📋 خطوات البدء السريع - Quick Start (3 خطوات فقط!)
═══════════════════════════════════════════════════════════════════

1️⃣ إضافة اختصار لسطح المكتب
   ────────────────────────────────────────────────────────────────
   انقر مرتين على:
   
   ✅ Add_License_Manager_To_Desktop.bat
   
   سيتم إنشاء اختصار "License Manager" على سطح المكتب


2️⃣ تشغيل التطبيق
   ────────────────────────────────────────────────────────────────
   اختر إحدى الطرق:
   
   ✅ انقر مرتين على "License Manager" في سطح المكتب (الأسهل)
   
   أو
   
   ✅ انقر مرتين على: Run_License_Manager.bat
   
   أو
   
   ✅ شغّل الأمر: python License_Manager_App.pyw


3️⃣ ابدأ الاستخدام!
   ────────────────────────────────────────────────────────────────
   التطبيق جاهز الآن - يمكنك:
   
   ➕ إنشاء تراخيص جديدة
   🔍 البحث عن التراخيص
   ✅ تفعيل/تعليق التراخيص
   ❌ حذف التراخيص
   📊 عرض الإحصائيات
   📤 تصدير التراخيص


═══════════════════════════════════════════════════════════════════
🎯 الأوامر الأساسية داخل التطبيق - Basic Commands
═══════════════════════════════════════════════════════════════════

1. ➕ إنشاء ترخيص جديد
   • انقر "➕ إنشاء ترخيص جديد"
   • أدخل اسم الشركة
   • أدخل المدة (بالأيام)
   • انقر "إنشاء"

2. 🔍 البحث عن ترخيص
   • انقر "🔍 البحث عن ترخيص"
   • أدخل المفتاح أو اسم الشركة
   • انقر "بحث"

3. ✅ تفعيل ترخيص
   • انقر "✅ تفعيل ترخيص"
   • أدخل مفتاح الترخيص
   • انقر "تفعيل"

4. ⏸️ تعليق ترخيص
   • انقر "⏸️ تعليق ترخيص"
   • أدخل مفتاح الترخيص
   • انقر "تعليق"

5. ❌ حذف ترخيص
   • انقر "❌ حذف ترخيص"
   • أدخل مفتاح الترخيص
   • أكد الحذف

6. 📊 عرض الإحصائيات
   • انقر "📊 عرض الإحصائيات"

7. 🔄 تحديث القائمة
   • انقر "🔄 تحديث القائمة"

8. 📤 تصدير التراخيص
   • انقر "📤 تصدير التراخيص"

🆘 حل المشاكل - Troubleshooting
═══════════════════════════════════════════════════════════════════

❌ التطبيق لا يفتح
   → تحقق من Python: python --version
   → شغّل بوضع debug: python License_Manager_App.pyw

❌ لا توجد تراخيص
   → أنشئ ترخيص جديد من التطبيق

❌ الاختصار لا يعمل
   → شغّل: Add_License_Manager_To_Desktop.bat
