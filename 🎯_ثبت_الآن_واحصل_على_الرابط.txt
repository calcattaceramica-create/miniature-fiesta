الخطوة 1️⃣: افتح Render.com
───────────────────────────

1. افتح الرابط التالي:
   👉 https://render.com

2. اضغط "Get Started for Free"

3. سجل الدخول بـ GitHub:
   - اضغط "Sign in with GitHub"
   - أدخل بيانات GitHub الخاصة بك
   - اضغط "Authorize Render"

4. ✅ تم! لديك الآن حساب مجاني


الخطوة 2️⃣: إنشاء Web Service
───────────────────────────

1. في Render Dashboard، اضغط "New +"

2. اختر "Web Service"

3. اضغط "Connect GitHub"

4. ابحث عن Repository:
   calcattaceramica-create/miniature-fiesta

5. اضغط "Connect" بجانب اسم Repository


الخطوة 3️⃣: إعداد المشروع
───────────────────────────

Render سيكتشف ملف render.yaml تلقائياً! ✅

ستظهر لك الإعدادات التالية:

   Name: ded-erp-system
   Region: Frankfurt
   Branch: main
   Build Command: pip install -r requirements.txt
   Start Command: gunicorn --bind 0.0.0.0:$PORT --workers 2 --timeout 120 run:app

✅ كل شيء صحيح! فقط اضغط "Create Web Service"


الخطوة 4️⃣: انتظر البناء
───────────────────────────

Render سيبدأ ببناء المشروع تلقائياً:

1. ستظهر لك شاشة "Deploying..."

2. انتظر 3-5 دقائق

3. ستظهر رسالة "Live" باللون الأخضر ✅


الخطوة 5️⃣: احصل على الرابط!
───────────────────────────

1. في أعلى الصفحة، ستجد الرابط:
   https://ded-erp-system.onrender.com

2. انسخ الرابط

3. افتحه في المتصفح

4. ✅ تم! المشروع يعمل الآن!

المشكلة 1: "Build Failed"
───────────────────────────

الحل:
1. تحقق من Logs في Render
2. تأكد من أن Python 3.11 محدد
3. أخبرني بالخطأ وسأساعدك


المشكلة 2: "Application Error"
───────────────────────────

الحل:
1. تحقق من Logs
2. تأكد من أن DATABASE_URL موجود في Environment
3. أخبرني بالخطأ وسأساعدك


المشكلة 3: الصفحة بطيئة
───────────────────────────

هذا طبيعي! ✅

- عند الزيارة الأولى، قد يأخذ 30-60 ثانية للتشغيل
- بعدها سيكون سريعاً
