                        🚀  كيفية الوصول من أجهزة أخرى

═══════════════════════════════════════════════════════════════════════════════════


┌───────────────────────────────────────────────────────────────────────────────┐
│  1️⃣  تأكد من تشغيل التطبيق على الجهاز الرئيسي                               │
│      Make sure the app is running on the main computer                       │
└───────────────────────────────────────────────────────────────────────────────┘

  ✅  افتح الواجهة على الجهاز الرئيسي
      Open the launcher on main computer

  ✅  اضغط زر "▶ تشغيل التطبيق"
      Click "▶ Start Application" button

  ✅  تأكد من ظهور "● التطبيق يعمل" باللون الأخضر
      Ensure "● Application Running" shows in green


┌───────────────────────────────────────────────────────────────────────────────┐
│  2️⃣  تأكد من اتصال الأجهزة بنفس الشبكة                                      │
│      Ensure devices are on the same network                                  │
└───────────────────────────────────────────────────────────────────────────────┘

  ⚠️  يجب أن تكون جميع الأجهزة متصلة بنفس شبكة WiFi أو نفس الراوتر
      All devices must be connected to the same WiFi or router


┌───────────────────────────────────────────────────────────────────────────────┐
│  3️⃣  افتح المتصفح على الجهاز الآخر                                          │
│      Open browser on the other device                                        │
└───────────────────────────────────────────────────────────────────────────────┘

  📱  من الموبايل - From Mobile:
      ═══════════════════════════════
      افتح أي متصفح (Chrome, Safari, Firefox)
      Open any browser (Chrome, Safari, Firefox)

  💻  من الكمبيوتر - From Computer:
      ═══════════════════════════════
      افتح أي متصفح (Chrome, Edge, Firefox)
      Open any browser (Chrome, Edge, Firefox)

  📟  من التابلت - From Tablet:
      ═══════════════════════════════
      افتح أي متصفح
      Open any browser


┌───────────────────────────────────────────────────────────────────────────────┐
│  4️⃣  اكتب الرابط في المتصفح                                                 │
│      Type the URL in the browser                                             │
└───────────────────────────────────────────────────────────────────────────────┘

  اكتب في شريط العنوان:
  Type in the address bar:

      ┌─────────────────────────────────────────────────────────────┐
      │  http://192.168.217.169:5000                                │
      └─────────────────────────────────────────────────────────────┘

  ⚠️  تأكد من كتابة الرابط بالضبط كما هو
      Make sure to type the URL exactly as shown


┌───────────────────────────────────────────────────────────────────────────────┐
│  5️⃣  سجل الدخول                                                              │
│      Login                                                                    │
└───────────────────────────────────────────────────────────────────────────────┘

  استخدم نفس بيانات الدخول:
  Use the same login credentials:

      👤  اسم المستخدم - Username:  admin
      🔑  كلمة المرور - Password:    admin123

                              ⚠️  ملاحظات مهمة جداً

  🔥  الجدار الناري - Firewall:
      ═══════════════════════════
      قد تحتاج لفتح المنفذ 5000 في جدار الحماية
      You may need to open port 5000 in firewall

      لفتح المنفذ في Windows:
      To open port in Windows:

      1. افتح Windows Defender Firewall
      2. اضغط "Advanced Settings"
      3. اضغط "Inbound Rules" → "New Rule"
      4. اختر "Port" → Next
      5. اكتب 5000 → Next
      6. اختر "Allow the connection" → Next
      7. سمّها "DED System" → Finish


  🔒  الأمان - Security:
      ═══════════════════
      ⚠️  هذا الرابط يعمل فقط على الشبكة المحلية
          This URL works only on local network

      ⚠️  لا تشارك الرابط خارج شبكتك
          Don't share the URL outside your network

      ⚠️  غيّر كلمة المرور الافتراضية للأمان
          Change default password for security


  📡  عنوان IP قد يتغير - IP May Change:
      ═══════════════════════════════════
      ⚠️  إذا أعدت تشغيل الراوتر، قد يتغير عنوان IP
          If you restart router, IP address may change

      ⚠️  في هذه الحالة، افتح الواجهة لمعرفة IP الجديد
          In this case, open launcher to see new IP
