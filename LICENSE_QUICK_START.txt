                        🚀 البدء السريع - Quick Start
═══════════════════════════════════════════════════════════════════════════════════


الخطوة 1️⃣: تثبيت جداول التراخيص
────────────────────────────────────

    python add_license_tables.py


الخطوة 2️⃣: إنشاء ترخيص لعميل جديد
────────────────────────────────────

    python create_client_license.py

    سيطلب منك:
    - اسم العميل
    - اسم المستخدم
    - كلمة المرور
    - نوع الترخيص (تجريبي، شهري، سنوي، دائم)


الخطوة 3️⃣: إرسال المعلومات للعميل
────────────────────────────────────

    بعد إنشاء الترخيص، ستحصل على:

    🔑 مفتاح الترخيص: XXXX-XXXX-XXXX-XXXX
    👤 اسم المستخدم: client1
    🔑 كلمة المرور: Pass@123

    أرسل هذه المعلومات للعميل


الخطوة 4️⃣: إدارة التراخيص
────────────────────────────

    python manage_licenses.py

    يمكنك:
    - عرض جميع التراخيص
    - تعليق ترخيص (إيقاف مؤقت)
    - إلغاء تعليق ترخيص
    - إلغاء ترخيص نهائياً
    - تمديد ترخيص


الخطوة 5️⃣: تفعيل فحص الترخيص (اختياري)
────────────────────────────────────────────

    في ملف app/__init__.py، قم بإلغاء التعليق عن:

    from app.license_middleware import init_license_middleware
    init_license_middleware(app)


═══════════════════════════════════════════════════════════════════════════════════
                        📋 أوامر سريعة - Quick Commands
═══════════════════════════════════════════════════════════════════════════════════


✅ إنشاء ترخيص جديد:
   python create_client_license.py


✅ عرض جميع التراخيص:
   python manage_licenses.py
   ثم اختر: 1


✅ تعليق ترخيص (إيقاف مؤقت):
   python manage_licenses.py
   ثم اختر: 3


✅ إلغاء تعليق ترخيص:
   python manage_licenses.py
   ثم اختر: 4


✅ تمديد ترخيص:
   python manage_licenses.py
   ثم اختر: 6


✅ إلغاء ترخيص نهائياً:
   python manage_licenses.py
   ثم اختر: 5


═══════════════════════════════════════════════════════════════════════════════════
                        🎯 سيناريوهات شائعة - Common Scenarios
═══════════════════════════════════════════════════════════════════════════════════


📌 السيناريو 1: عميل جديد يريد التطبيق
────────────────────────────────────────

    1. python create_client_license.py
    2. أدخل معلومات العميل
    3. احصل على اليوزر والباسورد
    4. أرسلها للعميل


📌 السيناريو 2: العميل لم يدفع
────────────────────────────────

    1. python manage_licenses.py
    2. اختر: 3 (تعليق ترخيص)
    3. أدخل رقم الترخيص
    4. أدخل السبب: "عدم السداد"
    5. ✅ التطبيق متوقف للعميل


📌 السيناريو 3: العميل دفع
────────────────────────────

    1. python manage_licenses.py
    2. اختر: 4 (إلغاء تعليق ترخيص)
    3. أدخل رقم الترخيص
    4. ✅ التطبيق يعمل مرة أخرى


📌 السيناريو 4: تمديد الاشتراك
────────────────────────────────

    1. python manage_licenses.py
    2. اختر: 6 (تمديد ترخيص)
    3. أدخل رقم الترخيص
    4. أدخل عدد الأيام (مثلاً: 365)
    5. ✅ تم التمديد


📌 السيناريو 5: إنهاء العقد نهائياً
────────────────────────────────────

    1. python manage_licenses.py
    2. اختر: 5 (إلغاء ترخيص نهائياً)
    3. أدخل رقم الترخيص
    4. تأكيد: y
    5. ✅ الترخيص ملغى نهائياً
