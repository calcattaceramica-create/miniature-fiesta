الخطوة 1️⃣: افتح Render.com
───────────────────────────

👉 https://render.com

1. اضغط "Get Started for Free"
2. اختر "Sign in with GitHub"
3. أدخل بيانات GitHub
4. اضغط "Authorize Render"


الخطوة 2️⃣: إنشاء Web Service
───────────────────────────

1. اضغط "New +" في الأعلى
2. اختر "Web Service"
3. اضغط "Connect GitHub"
4. ابحث عن: calcattaceramica-create/miniature-fiesta
5. اضغط "Connect"


الخطوة 3️⃣: إعداد المشروع
───────────────────────────

✅ Render سيكتشف ملف render.yaml تلقائياً!

تأكد من الإعدادات:
   Name: ded-erp-system
   Region: Frankfurt
   Branch: main

ثم اضغط "Create Web Service"


الخطوة 4️⃣: انتظر البناء
───────────────────────────

1. ستظهر شاشة "Deploying..."
2. انتظر 3-5 دقائق
3. ستظهر "Live" باللون الأخضر ✅


الخطوة 5️⃣: احصل على الرابط!
───────────────────────────

الرابط المتوقع:
👉 https://ded-erp-system.onrender.com

بيانات الدخول:
   Username: admin
   Password: admin123
