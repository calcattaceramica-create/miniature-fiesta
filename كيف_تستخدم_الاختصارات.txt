🚀 كيفية تشغيل التطبيق الرئيسي
How to Launch Main Application
═══════════════════════════════════════════════════════════════════════════════

الخطوة 1 / Step 1:
────────────────────────────────────────────────────────────────────────────
اذهب إلى سطح المكتب
Go to Desktop

الخطوة 2 / Step 2:
────────────────────────────────────────────────────────────────────────────
ابحث عن الاختصار: "DED ERP System"
Find the shortcut: "DED ERP System"

الخطوة 3 / Step 3:
────────────────────────────────────────────────────────────────────────────
انقر نقراً مزدوجاً على الاختصار
Double-click the shortcut

الخطوة 4 / Step 4:
────────────────────────────────────────────────────────────────────────────
ستظهر نافذة سوداء (CMD) - لا تغلقها!
A black window (CMD) will appear - DON'T CLOSE IT!

الخطوة 5 / Step 5:
────────────────────────────────────────────────────────────────────────────
انتظر 2-3 ثواني
Wait 2-3 seconds

الخطوة 6 / Step 6:
────────────────────────────────────────────────────────────────────────────
سيفتح المتصفح تلقائياً على: http://localhost:5000
Browser will open automatically at: http://localhost:5000

الخطوة 7 / Step 7:
────────────────────────────────────────────────────────────────────────────
سجل الدخول:
Login:
   اسم المستخدم / Username: admin
   كلمة المرور / Password: admin123

🔑 كيفية تشغيل نظام التراخيص
How to Launch License Manager
═══════════════════════════════════════════════════════════════════════════════

الخطوة 1 / Step 1:
────────────────────────────────────────────────────────────────────────────
اذهب إلى سطح المكتب
Go to Desktop

الخطوة 2 / Step 2:
────────────────────────────────────────────────────────────────────────────
ابحث عن الاختصار: "License Manager"
Find the shortcut: "License Manager"

الخطوة 3 / Step 3:
────────────────────────────────────────────────────────────────────────────
انقر نقراً مزدوجاً على الاختصار
Double-click the shortcut

الخطوة 4 / Step 4:
────────────────────────────────────────────────────────────────────────────
ستظهر نافذة نظام التراخيص مباشرة
License Manager window will appear directly

الخطوة 5 / Step 5:
────────────────────────────────────────────────────────────────────────────
اختر من القائمة:
Choose from menu:
   1 - إنشاء ترخيص جديد / Create new license
   2 - عرض التراخيص / View licenses
   3 - التحقق من ترخيص / Validate license
   4 - خروج / Exit

Very Important Notes
═══════════════════════════════════════════════════════════════════════════════

❗ النافذة السوداء (CMD) يجب أن تبقى مفتوحة
   The black window (CMD) MUST stay open

❗ لا تغلق النافذة السوداء أثناء استخدام التطبيق
   Don't close the black window while using the application

❗ إذا أغلقت النافذة السوداء، سيتوقف التطبيق
   If you close the black window, the application will stop

❗ لإيقاف التطبيق: اضغط CTRL+C في النافذة السوداء
   To stop the application: Press CTRL+C in the black window

❗ إذا لم يفتح المتصفح تلقائياً، افتحه يدوياً واذهب إلى:
   If browser doesn't open automatically, open it manually and go to:
   http://localhost:5000

🔧 حل المشاكل الشائعة
Troubleshooting
═══════════════════════════════════════════════════════════════════════════════

المشكلة 1: النافذة السوداء تظهر وتختفي بسرعة
Problem 1: Black window appears and disappears quickly
────────────────────────────────────────────────────────────────────────────
الحل / Solution:
• شغّل ملف: test_launch.bat
• سيظهر لك الخطأ بالتفصيل
• Run file: test_launch.bat
• It will show you the detailed error

المشكلة 2: المتصفح لا يفتح تلقائياً
Problem 2: Browser doesn't open automatically
────────────────────────────────────────────────────────────────────────────
الحل / Solution:
• افتح المتصفح يدوياً
• اذهب إلى: http://localhost:5000
• Open browser manually
• Go to: http://localhost:5000

المشكلة 3: رسالة خطأ "Port already in use"
Problem 3: Error message "Port already in use"
────────────────────────────────────────────────────────────────────────────
الحل / Solution:
• التطبيق يعمل بالفعل!
• افتح المتصفح واذهب إلى: http://localhost:5000
• Application is already running!
• Open browser and go to: http://localhost:5000

المشكلة 4: رسالة خطأ "Virtual environment not found"
Problem 4: Error message "Virtual environment not found"
────────────────────────────────────────────────────────────────────────────
الحل / Solution:
• شغّل ملف: setup.bat
• انتظر حتى ينتهي التثبيت
• Run file: setup.bat
• Wait until installation completes
