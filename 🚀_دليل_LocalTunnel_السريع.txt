🎯 ما هو LocalTunnel؟
────────────────────────────────────────────────────────────────────────────────

LocalTunnel هو أداة مجانية تماماً تتيح لك الوصول إلى تطبيقك من أي مكان في العالم!

✅ مجاني تماماً بدون حدود زمنية
✅ رابط ثابت (يمكنك اختياره)
✅ لا يحتاج تسجيل أو حساب
✅ HTTPS آمن
✅ سهل الاستخدام


═══════════════════════════════════════════════════════════════════════════════
                    📋 الخطوات (أول مرة فقط)
═══════════════════════════════════════════════════════════════════════════════


الخطوة 1️⃣: تثبيت Node.js
────────────────────────────────────────────────────────────────────────────────

1. اذهب إلى: https://nodejs.org/
2. حمّل النسخة LTS (الموصى بها)
3. ثبّت البرنامج (Next → Next → Install)
4. أعد تشغيل الكمبيوتر (اختياري)


الخطوة 2️⃣: تثبيت LocalTunnel
────────────────────────────────────────────────────────────────────────────────

1. افتح CMD (موجه الأوامر)
2. اكتب: npm install -g localtunnel
3. اضغط Enter وانتظر
4. ✅ تم!


الخطوة 3️⃣: تشغيل النظام
────────────────────────────────────────────────────────────────────────────────

1. انقر نقرتين على: 🏆_LocalTunnel_مجاني_بدون_حدود.bat
2. اختر "رابط مخصص" (الخيار 2)
3. اكتب اسماً للرابط (مثال: myapp)
4. ✅ رابطك جاهز: https://myapp.loca.lt

من الكمبيوتر:
────────────────────────────────────────────────────────────────────────────────

1. افتح المتصفح (Chrome, Firefox, Edge)
2. اكتب الرابط: https://myapp.loca.lt
3. عند أول زيارة: اضغط "Click to Continue"
4. سجل الدخول:
   - Username: admin
   - Password: admin123


من الهاتف:
────────────────────────────────────────────────────────────────────────────────

1. افتح المتصفح على الهاتف
2. اكتب الرابط: https://myapp.loca.lt
3. عند أول زيارة: اضغط "Click to Continue"
4. سجل الدخول:
   - Username: admin
   - Password: admin123


من أي مكان في العالم:
────────────────────────────────────────────────────────────────────────────────

✅ نفس الخطوات!
✅ الرابط يعمل من أي مكان!


═══════════════════════════════════════════════════════════════════════════════
                    ⚠️  ملاحظات مهمة
═══════════════════════════════════════════════════════════════════════════════


عند أول زيارة:
────────────────────────────────────────────────────────────────────────────────

⚠️  قد يظهر لك صفحة "Friendly Reminder"
⚠️  هذا إجراء أمني طبيعي من LocalTunnel
✅ فقط اضغط "Click to Continue" أو "Continue"
✅ ستفتح الصفحة مباشرة


النوافذ:
────────────────────────────────────────────────────────────────────────────────

⚠️  لا تغلق النافذتين (الخضراء والزرقاء)
⚠️  النافذة الخضراء: السيرفر المحلي
⚠️  النافذة الزرقاء: LocalTunnel (الرابط العام)


الأمان:
────────────────────────────────────────────────────────────────────────────────

✅ HTTPS آمن
✅ البيانات مشفرة
✅ غيّر كلمة المرور بعد أول تسجيل دخول
✅ لا تشارك الرابط مع أشخاص غير موثوقين


الاستخدام:
────────────────────────────────────────────────────────────────────────────────

✅ الرابط يعمل فقط عندما يكون السيرفر يعمل
✅ إذا أغلقت النوافذ، الرابط لن يعمل
✅ لإعادة التشغيل: شغّل الملف مرة أخرى


═══════════════════════════════════════════════════════════════════════════════
                    🔧 حل المشاكل
═══════════════════════════════════════════════════════════════════════════════


المشكلة: "Node.js غير مثبت"
────────────────────────────────────────────────────────────────────────────────

الحل:
1. ثبّت Node.js من: https://nodejs.org/
2. أعد تشغيل الكمبيوتر
3. جرب مرة أخرى


المشكلة: "فشل التثبيت"
────────────────────────────────────────────────────────────────────────────────

الحل:
1. شغّل CMD كمسؤول (Run as Administrator)
2. اكتب: npm install -g localtunnel
3. جرب مرة أخرى


المشكلة: "الرابط لا يعمل"
────────────────────────────────────────────────────────────────────────────────

الحل:
1. تأكد أن النافذتين (الخضراء والزرقاء) مفتوحتين
2. تأكد أن السيرفر يعمل
3. جرب الرابط مرة أخرى


المشكلة: "Click to Continue" يظهر كل مرة
────────────────────────────────────────────────────────────────────────────────

الحل:
✅ هذا طبيعي!
✅ LocalTunnel يطلب هذا للأمان
✅ فقط اضغط "Continue"

────────────────────────────────────────────────────────────────────────────────
