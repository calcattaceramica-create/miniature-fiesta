                        🚀 البدء السريع - Quick Start
═══════════════════════════════════════════════════════════════════════════════════


الخطوة 1️⃣: إنشاء ترخيص لعميل جديد
────────────────────────────────────────

    python create_client_license.py

    سيطلب منك:
    - اسم العميل: شركة ABC
    - اسم المستخدم: abc_admin
    - كلمة المرور: Abc@2026
    - نوع الترخيص: سنوي (365 يوم)

    ستحصل على:
    🔑 مفتاح الترخيص: XXXX-XXXX-XXXX-XXXX
    👤 اسم المستخدم: abc_admin
    🔑 كلمة المرور: Abc@2026


الخطوة 2️⃣: إرسال المعلومات للعميل
────────────────────────────────────

    أرسل للعميل:
    - رابط التطبيق
    - اسم المستخدم
    - كلمة المرور
    - مدة الترخيص


الخطوة 3️⃣: إدارة التراخيص
────────────────────────────

    python manage_licenses.py

    يمكنك:
    1. عرض جميع التراخيص
    2. عرض تفاصيل ترخيص
    3. تعليق ترخيص (إيقاف مؤقت)
    4. إلغاء تعليق ترخيص
    5. إلغاء ترخيص نهائياً
    6. تمديد ترخيص


═══════════════════════════════════════════════════════════════════════════════════
                        💡 سيناريوهات عملية - Practical Scenarios
═══════════════════════════════════════════════════════════════════════════════════


📌 السيناريو 1: عميل جديد يريد التطبيق
────────────────────────────────────────────

    1. python create_client_license.py
    2. أدخل معلومات العميل
    3. احصل على اليوزر والباسورد
    4. أرسلها للعميل
    5. ✅ العميل يستخدم التطبيق


📌 السيناريو 2: العميل لم يدفع
────────────────────────────────

    1. python manage_licenses.py
    2. اختر: 3 (تعليق ترخيص)
    3. أدخل رقم الترخيص
    4. السبب: "عدم السداد"
    5. ✅ التطبيق متوقف للعميل


📌 السيناريو 3: العميل دفع
────────────────────────────

    1. python manage_licenses.py
    2. اختر: 4 (إلغاء تعليق ترخيص)
    3. أدخل رقم الترخيص
    4. ✅ التطبيق يعمل مرة أخرى


📌 السيناريو 4: تمديد الاشتراك
────────────────────────────────

    1. python manage_licenses.py
    2. اختر: 6 (تمديد ترخيص)
    3. أدخل رقم الترخيص
    4. أدخل عدد الأيام: 365
    5. ✅ تم التمديد لسنة إضافية


📌 السيناريو 5: إنهاء العقد نهائياً
────────────────────────────────────────

    1. python manage_licenses.py
    2. اختر: 5 (إلغاء ترخيص نهائياً)
    3. أدخل رقم الترخيص
    4. تأكيد: y
    5. ✅ الترخيص ملغى نهائياً


═══════════════════════════════════════════════════════════════════════════════════
                        🔒 تفعيل فحص الترخيص التلقائي (اختياري)
                        Enable Automatic License Check (Optional)
═══════════════════════════════════════════════════════════════════════════════════


لتفعيل فحص الترخيص التلقائي:

1. افتح ملف: app/__init__.py

2. ابحث عن هذه الأسطر:
   # Initialize license middleware (optional - uncomment to enable)
   # from app.license_middleware import init_license_middleware
   # init_license_middleware(app)

3. قم بإزالة علامة # لتصبح:
   # Initialize license middleware
   from app.license_middleware import init_license_middleware
   init_license_middleware(app)

4. أعد تشغيل التطبيق

5. ✅ الآن يتم فحص الترخيص تلقائياً كل 5 دقائق
