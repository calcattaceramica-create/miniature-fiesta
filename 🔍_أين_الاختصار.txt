🔍 كيف تجده
How to Find It
═══════════════════════════════════════════════════════════════════════════════

الطريقة 1 / Method 1:
────────────────────────────────────────────────────────────────────────────

   1. اضغط على مفتاح Windows + D
      Press Windows + D key
      
   2. سيظهر سطح المكتب
      Desktop will appear
      
   3. ابحث عن ملف باسم: DED System
      Look for file named: DED System
      
   4. الأيقونة: 📁 مجلد أزرق
      Icon: 📁 Blue folder

الطريقة 2 / Method 2:
────────────────────────────────────────────────────────────────────────────

   1. افتح File Explorer (مستكشف الملفات)
      Open File Explorer
      
   2. اذهب إلى: Desktop
      Go to: Desktop
      
   3. ابحث عن: DED System
      Look for: DED System

الطريقة 3 / Method 3:
────────────────────────────────────────────────────────────────────────────

   1. اضغط Windows + R
      Press Windows + R
      
   2. اكتب: shell:desktop
      Type: shell:desktop
      
   3. اضغط Enter
      Press Enter
      
   4. ابحث عن: DED System
      Look for: DED System

🚀 ماذا يحدث عند فتحه
What Happens When You Open It
═══════════════════════════════════════════════════════════════════════════════

عند النقر المزدوج على الاختصار:
When you double-click the shortcut:

   1. ستظهر واجهة رسومية جميلة
      A beautiful graphical interface will appear
      
   2. الواجهة تحتوي على 3 أزرار:
      The interface contains 3 buttons:
      
      🚀 تشغيل التطبيق الرئيسي (أخضر)
         Launch Main Application (Green)
      
      🔑 نظام إدارة التراخيص (أزرق)
         License Manager (Blue)
      
      🌐 فتح التطبيق في المتصفح (بنفسجي)
         Open in Browser (Purple)
      
   3. اختر الزر الذي تريده
      Choose the button you want
